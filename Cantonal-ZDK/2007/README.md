# XII Cantonal IT Competition ZDK (2007) 🏆

This folder contains my solutions to the problems from the **XII Cantonal IT Competition for High School Students**, held in Tešanj, Bosnia and Herzegovina, on April 21, 2007. 

The test cases for these algorithms are validated against the official competition test sets.

## 📂 Problem Descriptions

Here are the translated descriptions of the five tasks presented in the competition paper:

### Task 1: Bacteria Division (Population Growth)
Write a program for bacteria division. The input data loaded are the initial number of bacteria, the division factor by which the bacteria divide, and the upper limit of the number of bacteria. The time of bacteria division in one generation is one hour. After how many hours will the bacteria division exceed the upper limit of the number of bacteria for the first time?
*   **Example:** For inputs 1, 2, and 1023 the result is 10. 

### Task 2: Largest Odd Divisor
Load a natural number. Find and print its largest odd divisor.
*   **Example:** For input 10 result is 5. 

### Task 3: Point Position Relative to Two Circles
Write a program that will load the (x1,y1) coordinates of the first circle, (x2,y2) coordinates of the second circle, coordinates of a point (x3,y3), radius of the first circle d1, and radius of the second circle d2. Determine the position of the point with coordinates (x3,y3) relative to the circles, i.e., whether the point lies outside the circles, only in the first or only in the second circle, in their intersection, or at the point of contact.
*   **Example:** For input (x1,y1)=(8,8), (x2,y2)=(4,4), (x3,y3)=(3,3), d1= 2 and d2= 2 the result is: inside second circle. 

### Task 4: Day of the Week for January 1st
Write a program that will print which day (Monday, Tuesday, ... etc.) is the first of January for a loaded year in the range [1600-2008].
* **Example:** For the year 2001, the output is "Ponedeljak" (Monday).

### Task 5: Read and Sort K-Largest Numbers from a File
In the file `brojevi.txt`, there are `n` different natural numbers, each placed in a separate line. Write a program that will read from the file the `k` largest numbers sorted from largest to smallest.
*   **Example:** Unable to provide example since I do not have the original brojevi.txt file.
