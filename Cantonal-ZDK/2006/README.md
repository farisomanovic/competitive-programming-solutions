# XI Cantonal IT Competition ZDK (2006) 🏆

This folder contains my solutions to the problems from the **XI Cantonal IT Competition for High School Students**, held in Zenica, Bosnia and Herzegovina, on April 29, 2006.

## 📂 Problem Descriptions

Here are the translated descriptions of the five tasks presented in the competition paper:

### Task 1: Greatest Common Divisor (GCD) Calculator
Write a program to calculate and print the Greatest Common Divisor (NZD - Najveći Zajednički Djelilac) for two natural numbers read from the keyboard. 
*   **Example:** For inputs 6 and 15, the GCD is 3. `GCD(6, 15) = 3`. 

### Task 2: Find Prime Numbers in an Interval
Write a program to find and print all prime numbers within a given interval. The upper and lower bounds of the interval are entered from the keyboard. 
*   **Example:** For the input interval [10, 20], the prime numbers are 11, 13, 17, and 19. 

### Task 3: Fibonacci Sequence Member Check
Write a program that prints whether a natural number read from the keyboard is a member of the Fibonacci sequence. It is mandatory to print a message stating whether the number belongs to the Fibonacci sequence and what member it is in the sequence. 
*   **Example:** For input 5, the message is "DA" (YES) and the number is the 5th member of the Fibonacci sequence. 

### Task 4: Parity-Based Array Filtering
Read `n` members of an array of integers. If the sum of the members of the read array is even, print only the read even members of the array; if the sum is odd, print only the odd read members of the array; and if the sum of the read members of the array is 0 (zero), print all read members of the array. 
*   **Example:** For the array -1, -2, 3, 5, 7, the output is -2. 

### Task 5: Alphabetical Name Sorter
Write a program to enter 5 student names as 5 strings. Print them in alphabetical order. 
*   **Example:** For the input list "Vlado, Enes, Mia Martina, Zajko, Alen", the output is "Alen, Enes, Mia Martina, Vlado, Zajko".
