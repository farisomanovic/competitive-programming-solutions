# XV Cantonal IT Competition ZDK (2010) 🏆

This folder contains my solutions to the problems from the **XV Cantonal IT Competition for High School Students**, held at the Gymnasium "Musa Ćazim Ćatić" in Tešanj, Bosnia and Herzegovina, on March 27, 2010.

## 📂 Problem Descriptions

Here are the translated descriptions of the five algorithmic tasks presented in the competition paper:

### Task 1: Flight Landing Time
Write a program that will determine the plane's landing time in GMT (hour, minute, and second) based on the entered GMT time (hour, minute, and second) and the flight duration expressed in seconds.
* **Example:** For h=5 hours, m=5 minutes, s=5 seconds, and a flight duration of 18305 seconds, the landing time is: h=10 hours, m=10 minutes, and s=10 seconds.

### Task 2: Line Intersection
Load the coefficients of two lines $a_1, b_1$ and $a_2, b_2$. Write a program that will determine the mutual position of lines located in a plane. If the lines intersect, determine the point of intersection.
* **Example:** For $a_1=5, b_1=2$ and $a_2=2, b_2=5$, the solution is: The lines intersect at point P=(x,y)=(1,7).

### Task 3: Polygon Area
Write a program that will calculate the area of a polygon based on entered vertices in a coordinate system.
* **Example:** For a square with vertices A(2,0), B(0,2), C(-2,0), D(0,-2), the area is P = 8.

### Task 4: Next Calendar Date
Write a program that will, for an entered date (DD,MM,YYYY), print the following date (the next day). Note: Pay attention to leap years, dates that do not exist, and valid values for the next period.
* **Example:** For the entered date 29.02.2020, it should print tomorrow's date: 1.3.2020.

### Task 5: Permutable Prime Numbers
Write a program that will establish which three-digit prime numbers satisfy the following condition: permuting the digits of the prime three-digit number results in new prime three-digit numbers for any permutation. (Note: some digits of the three-digit number can be the same, for example, 101, so the permutations can be with repetition). Print all such numbers from all 6 prime numbers obtained by permuting their digits.
* **Example:** For the three-digit number abc, the permutations would be: abc, acb, bac, bca, cab, cba (all these numbers must be prime).
