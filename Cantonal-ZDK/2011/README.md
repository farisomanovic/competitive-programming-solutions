# XVI Cantonal IT Competition ZDK (2011) 🏆

This folder contains my solutions to the problems from the **XVI Cantonal IT Competition for High School Students**, held at the First Gymnasium in Zenica, Bosnia and Herzegovina, on April 16, 2011. 

The algorithms here are validated against the official competition test datasets.

## 📂 Problem Descriptions

Here are the translated descriptions of the five algorithmic tasks presented in the competition paper:

### Task 1: Min, Max, and Occurrences
Write a program to enter 3 natural numbers from the keyboard (they can be different, some can be equal, or all three can be equal). Based on them, print 4 pieces of data: 
a) the minimum member of the entered numbers, 
b) the number of minimum members, 
c) the maximum member, 
d) the number of maximum members.
* **Example:** For entered numbers 4, 4, 4, the output should be: a) min=4, b) min count=3, c) max=4, d) max count=3.

### Task 2: Right Triangle Legs
Write a program to input the hypotenuse `c` [cm] of a triangle ABC. The sides of the triangle are whole numbers. Print its legs `a` and `b`, if the triangle is a right triangle, or a message that the triangle is not a right triangle.
* **Example:** For an entered hypotenuse c=5, the legs are a=3 and b=4. For hypotenuse c=7, the triangle is not right-angled.

### Task 3: The Fly and Cyclists Distance
Write a program to calculate the distance a fly travels in the event that it flies from one cyclist to another. Cyclist A moves towards point B, and Cyclist B moves towards point A. The fly starts together with cyclist A towards cyclist B. When it reaches cyclist B, it returns to cyclist A and continues doing so until the cyclists meet. The fly flies faster than both cyclists. Input data: speeds of cyclists A, B, and the fly (v_a, v_b, and v_m) in [km/h], and the distance between points `d` in [km]. Speeds are constant and integer values.
* **Example:** For entered v_a=8, v_b=10, v_m=27, and d=15, the fly covers s=22.5 [km].

### Task 4: Maximum Prime Gap
Write a program to input a natural number `n` as the upper bound of the interval of natural numbers [1,n]. Establish and print the maximum difference between two adjacent prime numbers in the interval [1,n]. 
* **Example:** For entered n=30 (interval [1,30]), the max difference between adjacent prime numbers is max_dif = 6 (which is 29 - 23).

### Task 5: Shortest Distance to Origin
Write a program to input the endpoints of a line segment AB: A(x1,y1) and B(x2,y2), which can be in all 4 quadrants. Determine and print the shortest distance of the line segment AB from the coordinate origin O(0,0).
* **Example:** For entered segment AB with A(-1,-1) and B(-2,-2), the shortest distance is 1.41.
