# XIII Cantonal IT Competition ZDK (2008) 🏆

This folder contains my solutions to the problems from the **XIII Cantonal IT Competition for High School Students**, held at the Mixed Secondary Technical School in Tešanj, Bosnia and Herzegovina, on April 5, 2008.

## 📂 Problem Descriptions

Here are the translated descriptions of the five algorithmic tasks presented in the competition paper:

### Task 1: 3D Distance Calculator
Write a program to calculate the distance between two arbitrary points in a three-dimensional coordinate system. The input data loaded from the keyboard are the coordinates of points T1(x1, y1, z1) and T2(x2, y2, z2).
* **Example:** For T1(0,0,0) and T2(3,4,5), their mutual distance is d=7.07.

### Task 2: Quadratic Equation Solver
Write a program to solve the quadratic equation $ax^2+bx+c$. The input data loaded from the keyboard are the coefficients of the equation a, b, and c.
* **Example:** For a=2, b=2, c=2, the outputs are x1=6.5 and x2=-7.5.

### Task 3: Largest Odd Divisor of an Even Number
Write a program that will print the largest odd divisor of a loaded even natural number.
* **Example:** For n=48, the largest odd divisor is 3.

### Task 4: Count Unique Numbers in an Array
Write a program that prints the number of different (unique) numbers from an array of loaded natural numbers. The loaded array can have a maximum of 50 numbers.
* **Example:** For the entered array 1, 2, 3, 2, 1, the output is 3 different numbers.

### Task 5: Prime Numbers in [n, 2n] Interval
Write a program that will, for an entered natural number n<1000, print all prime numbers in the interval [n, 2n]. Note: prime numbers are those that have only two divisors, the number 1 and themselves. 
* **Example:** For the entered number n=10, the prime numbers from the interval [10, 20] are 11, 13, 17, and 19.
