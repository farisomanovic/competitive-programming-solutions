# XVII Cantonal IT Competition ZDK (2012) 🏆

This folder contains my solutions to the problems from the **XVII Cantonal IT Competition for High School Students**, held at the Mixed Secondary School in Tešanj, Bosnia and Herzegovina, on April 21, 2012.

## 📂 Problem Descriptions

Here are the translated descriptions of the five algorithmic tasks presented in the competition paper:

### Task 1: 3D Distance from Origin
Write a program through which the coordinates of point A(x,y,z) in a three-dimensional coordinate system will be entered. Calculate the distance of point A from the origin O(0,0,0).
* **Example:** Point A(0,1,0) is at a distance of d=1 from the origin O(0,0,0).

### Task 2: Sum of Odd Digits Greater Than 4
Write a program to enter a natural number less than 32726. Calculate the sum of the odd digits of that number that are greater than 4.
* **Example:** For the number n=32517, the sum of odd digits greater than 4 is 12, because the odd digits are (5 and 7)>4.

### Task 3: Last Two Digits of a Power of 2
Write a program that finds the last two digits of the number $2^n$ for n<32726. The number n is loaded from the keyboard.
* **Example:** For n=7, the solution is 28, because $2^7=128$, i.e., its last digits are 28.

### Task 4: Longest Palindrome Substring
Load a word as a string. Write a program that prints the longest substring that is a palindrome. (A palindrome is a word that reads the same from left to right and from right to left).
* **Example:** For the entered string "zadatak", the substring is "ada".

### Task 5: Max Difference in Groups of Three Primes
Write a program to enter a number n as the upper limit of the interval [2,n]. In the observed interval, we observe three adjacent prime numbers at a time (e.g., 2,3,5 then 3,5,7 and 5,7,11, etc. up to the upper limit n). Calculate the maximum difference between the largest and the smallest prime number from the thus observed groups of three prime numbers. All three adjacent prime numbers are in the mentioned interval.
* **Example:** For the entered number n=30, i.e., the observation interval [2,30], the maximum difference is 10, because for three adjacent numbers 19, 23, and 29: 29-19=10.
